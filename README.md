# This is a template setup for Lemlib for all 242 Teams to use
Plug in your port values and the project will do the rest.

## Features
- Auton selector
- Tracking Wheel Code
- Pre-Tuned Controllers
- Custom LCD (Replaces pros::lcd)
- Autonomous Position Tracking Task
